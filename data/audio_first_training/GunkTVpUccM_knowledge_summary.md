## Key Concepts Taught

### 1. Break of Structure (BOS) - Definition

**What It Is:**
- BOS indicates the CONTINUATION of forex trend
- **Bullish Market:** When price breaks previous swing high (bullish continuation)
- **Bearish Market:** When price breaks previous swing low (bearish continuation)

**Critical Understanding:**
- BOS is an **INDICATION** of trend continuation, NOT confirmation
- BOS is **timeframe-dependent** - each timeframe has its own BOS levels
  - Daily has its own BOS
  - H4 has its own BOS
  - H1 has its own BOS
  - M15 has its own BOS

**Key Question:** Is every break of a high or low a Break of Structure?
**Answer:** NO - requires validation via 2 rules

### 3. Rule 1: Price Must Take Out Inducement

**Requirement:** Price swing MUST sweep the inducement level before creating valid BOS

**CRITICAL CLARIFICATION:**
- For Rule 1, candle closing below inducement is **NOT compulsory**
- **Both valid:**
  - Candle wick sweep of inducement ✓
  - Candle body close below inducement ✓

**Validation:**
1. Identify inducement (pullback before current high)
2. Check if price took out this inducement
3. If YES → Rule 1 fulfilled, continue to Rule 2
4. If NO → FAKE BOS (invalid)

**Example:**
- Valid BOS: Price sweeps inducement, then creates new high
- Fake BOS: Price creates new high WITHOUT sweeping inducement

**Reference:** If unclear on inducement, watch Videos 3 & 4 (described as "forex life-changing")

---

### 4. Rule 2: Candle Body Must Close Above High / Below Low

**Requirement:** After taking out inducement, candle BODY must close ABOVE previous high (bullish) or BELOW previous low (bearish)

**CRITICAL: Body Close is REQUIRED**
- Unlike Rule 1, Rule 2 is STRICT about body close
- Wick-only break = NOT valid BOS = liquidity sweep

**Simple Scenario:**
- Candle body clearly closes above previous high → Valid BOS ✓

**Tricky Scenario:**
- Candle wicks above high but body closes below it
- Initial assessment: NOT valid BOS (liquidity sweep)
- **BUT:** Always wait for next candle
- If next candle's body closes above the sweeping candle's wick high → Valid BOS ✓

**Why Wait for Next Candle?**
- Need body close above previous highest point (even if that point is a wick)
- Next candle can convert liquidity sweep into valid BOS

### 5. Liquidity Sweep vs BOS

**Liquidity Sweep:**
- Wick breaks above high / below low
- Body closes back inside range
- Purpose: Hunt stop losses and pending orders
- Marking: Invalid BOS, mark as liquidity sweep

**Valid BOS:**
- Body closes beyond the level
- Confirms genuine break (not just liquidity hunt)
- Marking: Valid BOS

**Conversion Scenario:**
1. First candle: Wick sweep (liquidity sweep)
2. Wait for next candle
3. Next candle body closes above wick high → Converts to valid BOS

**Key Distinction:** Wick-only = liquidity sweep | Body close = BOS

---

### 6. Two Rules Summary & Validation Checklist

**Both Rules Must Be Fulfilled:**

**Rule 1:** Price must take out inducement
- Valid methods: Wick sweep OR body close (both acceptable)
- Body closing NOT compulsory for Rule 1

**Rule 2:** Candle body must close above high (bullish) / below low (bearish)
- Valid methods: Body close ONLY (wick not sufficient)
- Body closing IS compulsory for Rule 2

**Validation Checklist:**
1. ✓ Check Rule 1: Did price take out inducement?
2. ✓ Check Rule 2: Did candle body close above high / below low?
3. Both YES → Valid BOS (mark and trade)
4. Any NO → Fake/Invalid BOS (do not trade)

**Filtering Mechanism:**
- Rule 1 filters: Breaks without inducement sweep
- Rule 2 filters: Liquidity sweeps (wick-only breaks)
- Combined: Only genuine structure breaks pass both filters

## Step-by-Step BOS Validation Process

**For Every Potential BOS:**

1. **Always start with Rule 1**
   - Locate inducement level (pullback before high)
   - Check: Did price sweep this inducement?
   - If NO → STOP, it's fake BOS
   - If YES → Continue to step 2

2. **Check Rule 2**
   - Identify the highest candle before break (bullish) or lowest (bearish)
   - Check: Did candle body close above this high / below this low?
   - If YES → Valid BOS ✓
   - If NO (wick only) → Wait for next candle

3. **If wick-only sweep**
   - Mark as liquidity sweep initially
   - Wait for next new candle
   - If next candle body closes above wick high → Valid BOS ✓
   - If not → Remains liquidity sweep (invalid BOS)
